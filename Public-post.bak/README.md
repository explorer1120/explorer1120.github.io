* A：嘿 //是什么意思啊？
* B：嘿.
* A：呃 我问你//是什么意思？
* B：问吧.
* A：我刚才不是问了么？
* B：啊？
* A：你再看看记录...
* B：看完了.
* A：......所以//是啥？
* B：所以什么？
* A：你存心耍我呢吧？
* B：没有啊 你想问什么？

不断循环之后，A一气之下和B绝交，自己苦学程序。

N年之后，A终于修成正果，回想起B，又把聊天记录翻出来看，这时，他突然发现B没有耍他……

而他自己也不知道当年他问B的究竟是什么问题……

PS: // 在程序里是注释。。。

如何判断程序员在做什么

1.打字快且停顿:写代码中;

2键盘鼠标并用且不停点击:在 debug;

3不停滚轮或 pagedown:在查错;

4.不停按刷新/F5:脚本卡住了;

5.一动不动紧盯屏幕:程序抛出异常;

6.表情时而严肃时而轻松:产品上线测试中;

7.手放键盘上长时间未动:加班太多猝死了
