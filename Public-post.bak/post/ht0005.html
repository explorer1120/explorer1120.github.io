一、使用CSS改变文字大小

我们改变字体大小可以用<h1>-<h6>标签，但是我们有些时候需要特别大的字，怎么办呢？这时我们就要用到传说中的CSS

来，我们现在head里添加style标签，代码如下

<head>
	<meta charset="utf-8">
	<title>html文档</title>
	<style>
		
	</style>
</head>

CSS代码就写在style标签里

好，回归正题，如何调整字体大小呢？

在CSS中，可以使用"font-size"属性改变字体大小

我们先找到想要DIY的字，添加一个class

<p class="diy">这才够大</p>

然后在style标签里添加如下代码

.diy{
	font-size:x;
}

这里的x代表字体大小，单位是em(倍)或px(像素)

一般情况下，1em=16px，这个数值随父元素变化

所以，我们想让指定字体的大小达到普通字体的5倍，就需要...

<!DOCTYPE html>
<html>
<head>
	<meta charset="utf-8">
	<title>html.document</title>
	<style>
		.diy{
			font-size:5em;
		}
	</style>
</head>
<body>
	<p class="diy">大大大大大！</p>
</body>
</html>

结果如下

大大大大大！

二、用CSS改变文字颜色

在CSS中，改变字体颜色的属性是"color"

而color属性的值可以是...

好吧，一共有7种，我只说最常用的两种好了，HEX和RGB

HEX的格式是color:#66CCFF或#6CF

RGE的格式是color:rgb(102,204,255)

这两种都可以，但我个人比较喜欢HEX(十六进制)

扯了这么多，那怎样改变文字的颜色呢？比如把一串文字改成天依蓝？

首先，我们要确保那句话已经加上了class="diy"

然后，我们在CSS中添加

color:#66CCFF;

结果是这样

我是天依蓝！66CCFF！

PS:天依蓝的16进制色值是#66CCFF

三、用CSS改变字体

一般来说，网页的字体是由浏览器的默认字体决定的

但是有的人就不乐意了，他说："我喜欢微软雅黑，但浏览器的默认字体却是宋体，难不成还要换浏览器？"

我对那个人说，"恭喜你，猜错了，字体是可以通过'font-family属性'自定义的"

呃...虽然我刚才在演戏，但是可以通过css改字体却不假，我们在css里加入

.diy{
	font-family:Microsoft YaHei;
}

字体就会变成微软雅黑(Microsoft YaHei是微软雅黑的英文)

为了突出效果，我把微软雅黑改成微软雅黑Light，再演示

代码↓

<!DOCTYPE html>
<html>
<head>
	<meta charset="utf-8">
	<title>html.document</title>
	<style>
		.diy{
			font-family:Microsoft YaHei Light;
		}
	</style>
</head>
<body>
	<p class="diy">我是微软雅黑Light字体</p>
</body>
</html>

结果↓

我是微软雅黑Light字体
