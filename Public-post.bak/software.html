1.uTools

简介：uTools是一个极简、插件化、跨平台的现代桌面软件。通过自由选配丰富的插件，打造你得心应手的工具集合。
当你熟悉它后，能够为你节约大量时间，让你可以更加专注地改变世界。

使用提示：

①下载并安装uTools后，按下快捷键"alt+空格"即可唤出搜索框

②刚开始的uTools功能很少，你可以点击最右侧的图标安装想要的插件

③建(yi)议(ding)要安装本地搜索插件

下载地址：uTools，你的生产力工具集

2.Internet Download Manager(IDM)

简介：IDM是一款超级强大的下载工具，能和浏览无缝配合，自动抓取网页视频，完全可以满足日常下载需求。(bt、磁力除外)

注意事项：

不能用IDM下载重要文件，详见维基百科：IDM

提示：

①IDM会自动添加浏览器插件，以用于捕捉浏览器的下载内容，所以请不要删除IDM插件

②当提醒你需要注册时，千万不要输入网上的序列号！！

下载地址：IDM下载地址

3.bandizip

简介：Bandizip 是一款压缩软件，它支持zip、7z 和 rar 以及其它压缩格式。它拥有非常快速的压缩和解压缩的算法，适用于多核心压缩、快速拖放、高速压缩等功能。

说明：我之所以推荐bandizip而不是7zip，是因为7zip无法直接双击打开压缩包(反正我的电脑上是这样)，至于其他的国产软件大家可以自己测试，还可以结果发到我的QQ中(提示：虚拟机)

下载地址：Bandizip

6.Typora

Typora 一直是我认为桌面端笔记应用应有的终极形态。「它的功能之强大、设计之冷静、体验之美妙、理念之先进，我认为值得所有笔记应用厂商学习。」

Typora 首先是一个 Markdown 文本编辑器，它支持且仅支持 Markdown 语法的文本编辑。在 Typora 官网 上他们将 Typora 描述为 「A truly minimal markdown editor. 」。

一个文本编辑器，写得舒服是关键。我曾说过，「更有趣的是，一个笔记应用不会因为它支持 Markdown 语法而高级或易用很多。」，细枝末节处的人性化考虑才是最重要的。而 Typora 的编辑体验显然是经过深思熟虑设计的产物。

Typora 的图片插入功能是广受好评的。要知道，Markdown 原生不太注重图片插入的功能，但你可以在 Typora 中：

  • 直接使用 右键 - 复制 Ctrl + V 将网络图片、剪贴板图片复制到文档中
  • 拖动本地图片到文档中
